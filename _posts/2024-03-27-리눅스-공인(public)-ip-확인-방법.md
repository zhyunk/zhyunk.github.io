## curl ifconfig.me


`curl` 키워드에 `ifconfig.me`를 전달하면 내 서버의 public ip를 알 수 있다.

```bash
curl ifconfig.me
```

## 정리하게 된 계기

리눅스 서버 설치 후 외부에서 서버로 접속하기 위해 public ip를 알아야 했다.

public ip를 알아내는 다양한 방법이 있지만

그중에서도 네트워크 정보를 조회하는 `ifconfig`를 활용하는 방법이 제일 기억에 잘 남을 것 같아서 정리 ✏️
